## 1. 今日必读

- [Google closes deal to acquire Wiz](https://www.wiz.io/blog/google-closes-deal-to-acquire-wiz) (Hacker News)
  谷歌正式宣布完成以 320 亿美元收购云安全公司 Wiz 的交易，这是谷歌历史上规模最大的收购之一。此次收购将显著增强谷歌云在网络安全领域的竞争力，应对日益复杂的云环境威胁，同时也标志着云安全市场整合进入新阶段。

- [速腾聚创千线级激光雷达组合获百度萝卜快跑独家定点](https://36kr.com/newsflashes/3719360638170759) (36kr)

- [「格式塔科技」获1.5亿元天使轮融资](https://36kr.com/p/3718214514128514) (36kr)
  超声波脑机接口企业「格式塔科技」获 1.5 亿元天使轮融资，由国生资本、道彤投资联合领投。资金将用于首款产品研发与前期临床试验，超声波脑机接口因其无创、高分辨率优势，正成为继侵入式后的又一热门技术路线。

- [曝千问郁博文加入字节跳动](https://36kr.com/newsflashes/3719293584749959) (36kr)
  原阿里巴巴通义实验室 Qwen 大模型后训练负责人郁博文已正式加入字节跳动，担任 Seed 团队视觉模型与多模态交互团队后训练负责人。这一人事变动反映了头部大厂在多模态大模型领域的人才争夺战仍在加剧。

- [Many SWE-bench-Passing PRs would not be merged](https://metr.org/notes/2026-03-10-many-swe-bench-passing-prs-would-not-be-merged-into-main/) (Hacker News)

- [I was interviewed by an AI bot for a job](https://www.theverge.com/featured-video/892850/i-was-interviewed-by-an-ai-bot-for-a-job) (Hacker News)

## 3. 工具与深读

- [Temporal: A nine-year journey to fix time in JavaScript](https://bloomberg.github.io/js-blog/post/temporal/) (Hacker News)
  JavaScript 长期以来在处理日期和时间方面存在痛点，Temporal 提案历经九年终于迎来解决方案。文章详细介绍了 Temporal API 的设计理念与功能，旨在解决时区转换、日期解析等复杂问题，为开发者提供现代、可靠的时间处理工具。

- [BitNet: 100B Param 1-Bit model for local CPUs](https://github.com/microsoft/BitNet) (Hacker News)
  微软开源了 BitNet 项目，这是一个拥有 1000 亿参数的 1-bit 模型，专为本地 CPU 运行优化。该技术大幅降低了大模型的硬件门槛，使得在消费级设备上运行超大参数模型成为可能，为端侧 AI 普及提供了新路径。

- [Show HN: A tool that watches webpages and exposes changes as RSS](https://sitespy.app) (Hacker News)
  开发者因错过签证预约而开发了 Site Spy 工具，它能监控网页变化并生成 RSS 订阅源。对于缺乏更新通知功能的政府网站或监控页面，该工具提供了一种轻量级的自动化监控方案，帮助用户及时获取关键信息。

- [Rspress 2.0 发布：AI 原生文档](https://www.infoq.cn/article/Kf7qHzF2jGKLovi675il) (InfoQ)
  Rspress 2.0 正式发布，新版本主打 AI 原生文档特性，并带来了更快的启动速度与全新设计主题。该工具旨在提升开发者的文档编写与阅读体验，通过集成 AI 能力，让文档站点更加智能化、交互性更强。

- [Making WebAssembly a first-class language on the Web](https://hacks.mozilla.org/2026/02/making-webassembly-a-first-class-language-on-the-web/) (Hacker News)
  Mozilla 发布文章探讨如何让 WebAssembly 成为 Web 平台的一等公民。文章讨论了 Wasm 的最新进展及其在浏览器中的集成深度，预示着未来 Web 应用将能更高效地利用高性能计算能力，打破传统 JavaScript 的性能瓶颈。
