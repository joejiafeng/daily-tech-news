## 1. 今日必读

- [百度智能云发布全球首款手机龙虾应用](https://36kr.com/newsflashes/3719285232318083) (36kr)

- [速腾聚创千线级激光雷达组合获百度萝卜快跑独家定点](https://36kr.com/newsflashes/3719360638170759) (36kr)

- [「格式塔科技」获1.5亿元天使轮融资，加速超声波脑机接口临床开发](https://36kr.com/p/3718214514128514) (36kr)

- [Many SWE-bench-Passing PRs would not be merged](https://metr.org/notes/2026-03-10-many-swe-bench-passing-prs-would-not-be-merged-into-main/) (Hacker News)

- [用中转站的千万要小心，已经有人在 tg 大量购买用户数据用于训练](https://www.v2ex.com/t/1197381) (V2EX)

- [I was interviewed by an AI bot for a job](https://www.theverge.com/featured-video/892850/i-was-interviewed-by-an-ai-bot-for-a-job) (Hacker News)

- [Cursor 正经历生死存亡](https://www.infoq.cn/article/Gh36Y2kBs8ITeURPqwy2) (InfoQ)

## 3. 工具与深读

- [BitNet: 100B Param 1-Bit model for local CPUs](https://github.com/microsoft/BitNet) (Hacker News)
  微软开源了 BitNet b1.58，这是一个 100B 参数的 1-bit 大模型，专为本地 CPU 推理优化。该模型通过极致的量化技术大幅降低了大模型的硬件门槛，使得在普通消费级设备上运行千亿参数模型成为可能，极具实用价值。

- [Show HN: A context-aware permission guard for Claude Code](https://github.com/manuelschipper/nah/) (Hacker News)
  开发者发布了一款针对 Claude Code 的上下文感知权限守卫工具。它解决了在使用 AI 编程助手时可能出现的文件误删或密钥泄露风险，在保留便利性的同时提供了更精细的安全控制，适合注重代码安全的开发者使用。

- [Temporal: A nine-year journey to fix time in JavaScript](https://bloomberg.github.io/js-blog/post/temporal/) (Hacker News)
  Bloomberg 详细介绍了 JavaScript 新提案 Temporal 的九年发展历程。作为对现有 Date 对象的替代，Temporal 旨在彻底解决 JS 中时区处理和日期计算的顽疾，这篇长文是前端开发者深入理解现代日期处理标准的必读材料。

- [Show HN: I built a tool that watches webpages and exposes changes as RSS](https://sitespy.app) (Hacker News)
  一款名为 Site Spy 的工具上线，它能监控网页变化并将其转化为 RSS 订阅源。该工具解决了许多网站不提供 RSS 的问题，特别适合用于监控政府公告、价格变动或职位更新，是信息获取效率工具的有益补充。

- [CNN Explainer – Learn Convolutional Neural Network in Your Browser](https://poloclub.github.io/cnn-explainer/) (Hacker News)
  一个交互式网页工具，帮助用户在浏览器中直观学习卷积神经网络（CNN）。通过可视化展示每一层的特征图和参数变化，将抽象的深度学习概念具象化，非常适合初学者理解计算机视觉的核心原理。
