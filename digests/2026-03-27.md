# Daily Tech News | 2026-03-27

今日科技圈动态频出，苹果正式停产 Mac Pro 标志着一个时代的落幕，而速腾聚创首次实现单季盈利则彰显了激光雷达商业化的曙光。AI 领域，小米大模型登顶 OpenRouter 榜单，腾讯云发布 Agent 全景图强调工程落地，与此同时 LiteLLM 供应链攻击事件也为开发者敲响了安全警钟。

## 1. 今日必读

- [苹果正式停产 Mac Pro](https://9to5mac.com/2026/03/26/apple-discontinues-the-mac-pro/) (Hacker News)
  苹果已正式宣布停产 Mac Pro，这款曾被视为专业级台式机标杆的产品正式退出历史舞台。此举标志着苹果自研芯片转型战略的全面完成，同时也意味着专业工作站市场格局的重大转变，用户未来将更多依赖 Mac Studio 等产品线。

- [速腾聚创首次实现单季盈利，机器人产品销量激增](https://36kr.com/p/3739712915816457) (36kr)
  速腾聚创公布 2025 年财报，第四季度首次实现单季盈利，净利润 1.04 亿元。全年激光雷达销量约 91.2 万台，其中机器人产品销量同比激增 25 倍，这表明激光雷达技术正加速从自动驾驶向更广泛的机器人领域渗透，商业化落地成效显著。

- [雷军辞去金山云董事长](https://36kr.com/p/3740427947426049) (36kr)
  金山云发布公告称雷军已辞去董事长职务，不再担任公司执行董事。作为金山系的灵魂人物，雷军的卸任可能意味着金山云正在进行管理层的新老交替或战略聚焦，以应对当前云计算市场激烈的竞争环境。

- [小米 MiMo-V2-Pro 模型拿下 OpenRouter 周榜第一](https://36kr.com/p/3740507252670720) (36kr)
  小米 MiMo-V2-Pro 模型上线后迅速登顶 OpenRouter 周榜，成为该平台首个周 token 消耗量超 3 万亿的模型。该模型采用 MoE 架构，总参数 1 万亿，在编程领域市场占有率已超 30%，显示出小米在 AI 大模型领域的强劲竞争力。

- [My minute-by-minute response to the LiteLLM malware attack](https://futuresearch.ai/blog/litellm-attack-transcript/) (Hacker News)
  针对 LiteLLM 近期版本被植入恶意代码的事件，作者详细记录了分钟级的应急响应全过程。此次供应链攻击事件影响恶劣，再次警示开发者在依赖开源库时必须建立更严格的审查机制，警惕 PyPI 等包管理器中的潜在风险。

## 2. 趋势与解读

- [腾讯云 Agent 产品全景图首次亮相](https://36kr.com/p/3740603253571843) (36kr)
  腾讯云发布 Agent 产品全景图，升级 MaaS 平台为 TokenHub，并提出 Harness 工程能力是 AI 落地关键。这表明行业焦点正从单纯的模型参数竞争，转向包括工具调用、长记忆管理在内的系统工程能力构建，企业级 AI 应用进入深水区。

- [用得越多、失业越快？GitHub 大改 Copilot 规则](https://www.infoq.cn/article/4ODfuXhhcI9mAeYepXY0) (InfoQ)
  GitHub 修改 Copilot 规则，默认使用个人代码训练 AI，引发关于知识产权和隐私的广泛争议。这一变动折射出 AI 巨头在数据训练上的焦虑，也加剧了开发者社区对于代码所有权和平台信任危机的担忧，可能推动更多开发者转向自托管方案。

- [From 0% to 36% on Day 1 of ARC-AGI-3](https://www.symbolica.ai/blog/arc-agi-3) (Hacker News)
  Symbolica 在 ARC-AGI-3 基准测试中首日即达到 36% 的得分，这是一个衡量 AI 泛化推理能力的严苛测试。该进展表明，通过符号化方法或新型架构，AI 在解决未见过的逻辑推理问题上正取得突破，挑战了单纯依赖大规模神经网络的传统路径。

- [知行机器人连续完成两轮近亿元融资](https://36kr.com/p/3740523277517063) (36kr)
  知行机器人专注于灵巧手及具身智能核心零部件，连续完成两轮近亿元融资。随着具身智能热度攀升，资本正从大模型向机器人“身体”的关键硬件环节延伸，灵巧手作为执行末端成为新的投资热点，补齐了具身智能落地的最后一块拼图。

## 3. 工具与深读

- [DOOM Over DNS](https://github.com/resumex/doom-over-dns) (Hacker News)
  一个极具极客精神的开源项目，将经典游戏《毁灭战士》移植到 DNS 协议上运行。该项目展示了网络协议的非常规用法，为理解底层网络通信和在受限环境下传输数据提供了有趣的实验案例，适合网络编程爱好者研究。

- [Chroma Context-1: Training a Self-Editing Search Agent](https://www.trychroma.com/research/context-1) (Hacker News)
  Chroma 发布 Context-1，这是一个能够自我编辑的搜索智能体。它通过训练模型来动态修改搜索上下文，而非仅仅依赖检索，为构建更精准的 RAG（检索增强生成）系统提供了新思路，是 AI 工程化落地的重要参考。

- [Moving from GitHub to Codeberg, for lazy people](https://unterwaditzer.net/2025/codeberg.html) (Hacker News)
  一篇面向开发者的实用指南，介绍了如何从 GitHub 平滑迁移到开源平台 Codeberg。在 GitHub 商业化策略调整和 Copilot 争议背景下，寻找去中心化、尊重开源精神的替代方案成为部分开发者的新选择，文章提供了具体的迁移步骤。

- [HyperAgents: Self-referential self-improving agents](https://github.com/facebookresearch/hyperagents) (Hacker News)
  Meta 发布 HyperAgents，这是一个具有自我引用和自我改进能力的智能体框架。项目探索了让 AI 智能体自主优化自身代码和逻辑的可能性，是通往 AGI 路径上的重要探索性工具，适合关注 AI 前沿架构的研究者阅读。
