## 3. 工具与深读

- [Show HN: Oproxy – 在浏览器中检查和修改网络流量](https://github.com/sauravrao637/oproxy) (Hacker News)
  Oproxy 是一款开源工具，允许开发者直接在浏览器端检查和修改网络流量。对于前端调试、API 抓包分析及安全测试而言，这种轻量级无代理的方案极大降低了开发者的抓包门槛。

- [Zeroserve：可通过 eBPF 脚本化的零配置 Web 服务器](https://su3.io/posts/introducing-zeroserve) (Hacker News)
  Zeroserve 是一款零配置 Web 服务器，创新性地支持使用 eBPF 进行脚本化编程。开发者能在内核层面对网络请求进行高效处理与观测，为高性能服务端开发提供了极具想象力的新工具。

- [Sem：基于 Git 代码理解的新原语，而非 LSP](https://ataraxy-labs.github.io/sem/) (Hacker News)
  Sem 提出了一种代码理解的新范式，它不依赖传统的 LSP，而是在 Git 仓库之上构建实体关系。这为大规模代码库的语义分析、架构依赖梳理及 AI 代码理解提供了更宏观的底层支持。

- [Ntsc-rs：模拟模拟电视和 VHS 伪影的开源视频工具](https://ntsc.rs/) (Hacker News)
  Ntsc-rs 是一款开源视频模拟工具，能逼真还原老式模拟电视和 VHS 录像带的视觉瑕疵与复古质感。非常适合独立游戏开发、复古视频创作者使用，轻松为数字画面添加怀旧的模拟信号特效。

- [macOS 存储管理漫谈：去重是节约空间的最好方法](https://sspai.com/prime/story/macos-storage-management-deduplication) (sspai)
  文章探讨了 macOS 下的存储管理理念，指出相比用 LLM 盲目删文件，去重才是最安全有效的空间节约法。对于深受磁盘空间困扰的 Mac 用户，这是一篇极具实操价值的深度指南。
