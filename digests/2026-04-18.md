# Daily Tech News | 2026-04-18

科技行业今日迎来多项重磅变动，小米汽车成立来首次设立 CTO 岗位，智元机器人跨过量产门槛后转向软件生态建设。与此同时，OpenAI 战略调整导致 Sora 负责人离职，AI 商业化与成本控制成为行业新焦点。

## 1. 今日必读

- [小米汽车设立首位 CTO，胡峥楠出任集团副总裁](https://36kr.com/p/3770688595214857?f=rss) (36kr)
  小米集团宣布胡峥楠出任小米集团副总裁、汽车部 CTO，宋钢出任汽车部副总裁、参谋长。这是小米汽车自 2021 年成立以来首次设立 CTO 岗位，两位新高管均拥有超过二十年汽车行业经验，标志着小米汽车研发管理体系进一步成熟。

- [智元机器人量产破万台，转型 AI 大模型平台](https://36kr.com/p/3770721219035649?f=rss) (36kr)
  智元机器人跨过“机器人量产下线一万台”门槛后，在合作伙伴大会上重点转向软件生态，推出六大 AI 模型与七大生产力解决方案。这家由前华为“天才少年”创立的公司，正试图通过开放生态构建机器人行业的“安卓”地位。

- [OpenAI 调整战略，Sora 负责人宣布离职](https://36kr.com/p/3771676858827523?f=rss) (36kr)
  OpenAI 视频生成平台 Sora 负责人比尔·皮布尔斯宣布离职，公司正调整战略以减少“支线项目”，重点转向编程和企业应用。这一变动反映了头部 AI 公司在商业化变现压力下，开始收缩战线聚焦核心盈利业务。

- [“华为天才少年”创业项目行云集成电路获超 4 亿元融资](https://36kr.com/p/3770363954954755?f=rss) (36kr)
  北京行云集成电路宣布完成超 4 亿元 Pre-A 轮融资，致力于研发新一代推理芯片以重构显存成本。五源资本、赛富投资基金等领投，显示国产 GPGPU 赛道在算力需求爆发的背景下依然备受资本青睐。

- [NIST 宣布停止丰富化大部分 CVE 漏洞数据](https://risky.biz/risky-bulletin-nist-gives-up-enriching-most-cves/) (Hacker News)
  美国国家标准与技术研究院（NIST）表示将不再对大多数 CVE 漏洞进行详细的“丰富化”处理。这意味着安全社区可能面临漏洞元数据质量下降的风险，迫使私营安全厂商或开源社区填补这一关键基础设施的空白。

## 2. 趋势与解读

- [端侧 AI 成手机标配，荣耀谈 AI 手机未来方向](https://36kr.com/p/3770819743728131?f=rss) (36kr)
  随着各大厂商将 AI 能力下放至千元机，端侧 AI 已成为智能手机标配。荣耀 AI 专家李向东指出，虽然端侧 AI 方向尚未收敛，但手机因其普及率和算力优势，仍是目前 AI 落地的最佳载体，行业竞争正从硬件参数转向智能体验。

- [享刻智能获 1.5 亿元融资，深耕餐饮具身智能](https://36kr.com/p/3770651778761481?f=rss) (36kr)
  在众多企业扎堆人形机器人时，享刻智能选择深耕餐饮场景，完成 1.5 亿元 A 轮融资。创始人陈震主张以场景落地为核心推动具身智能商业化，这种务实路线为行业提供了差异化的发展样本，表明垂直场景落地正成为机器人赛道新趋势。

- [AI 智能体成本是否呈指数级上升引发讨论](https://www.tobyord.com/writing/hourly-costs-for-ai-agents) (Hacker News)
  文章探讨了 AI 智能体在处理复杂任务时，其运行成本可能随任务难度呈指数级上升的现象。这为当前火热的 Agent 热潮提供了冷静视角，提示企业在部署智能体时需重新评估算力成本与 ROI 的平衡，单纯追求智能程度可能导致成本失控。

- [Linux 内核或将接受 AI 代码，Linus 态度软化](https://www.infoq.cn/article/h4WPj7MeydhFDpsOqyCl) (InfoQ)
  随着 AI 编程能力的提升，Linux 内核社区对 AI 生成代码的态度正在发生变化。虽然 Linus Torvalds 曾持保留态度，但最新讨论显示，只要人类开发者对代码负责，AI 辅助生成的代码有望进入内核，这标志着开源社区对 AI 工具的接纳。

## 3. 工具与深读

- [Show HN: Smol machines – 亚秒级冷启动便携虚拟机](https://github.com/smol-machines/smolvm) (Hacker News)
  开发者展示了 Smol machines 项目，这是一种能够实现亚秒级冷启动的便携式虚拟机技术。该工具旨在解决传统虚拟机启动慢、体积大的痛点，为边缘计算和 Serverless 场景提供了新的轻量级解决方案，具有极高的实用价值。

- [PanicLock：合上 MacBook 即禁用 TouchID 的安全工具](https://github.com/paniclock/paniclock/) (Hacker News)
  受记者被迫解锁电脑事件启发，开发者推出了 PanicLock 工具。用户合上 MacBook 盖子即可自动禁用 TouchID，强制要求密码解锁，防止设备被强制指纹解锁，为敏感数据提供了物理层面的隐私保护，适合关注隐私安全的用户。

- [实测 Claude 4.7 新分词器对成本的影响](https://www.claudecodecamp.com/p/i-measured-claude-4-7-s-new-tokenizer-here-s-what-it-costs-you) (Hacker News)
  文章详细测量了 Claude 4.7 新版分词器的表现及其对使用成本的影响。分词器的效率直接关系到 API 调用的 Token 数量，开发者需关注这一底层变化，以优化应用在长文本处理中的费用支出，是使用 Anthropic 模型必读的技术分析。

- [Spring 团队专访：聚焦 Spring Framework 7 与 Boot 4](https://www.infoq.cn/article/z4msV9uzNy7CXYFC4K2J) (InfoQ)
  InfoQ 发布了对 Spring 团队的专访，深入探讨了即将到来的 Spring Framework 7 和 Spring Boot 4 的技术路线图。内容涵盖了框架对云原生、GraalVM 以及新 Java 版本的适配策略，对 Java 开发者规划未来技术栈具有重要参考价值。
