## 3. 工具与深读

- [Leanstral：开源的可信编码与形式化证明 Agent](https://mistral.ai/news/leanstral) (Hacker News)
  Mistral 推出 Leanstral，这是一个专注于代码生成和形式化证明的开源 Agent。它基于 Lean 4，旨在提高代码的可信度，为追求高可靠性的软件开发者提供了强大的辅助工具。

- [Oxyde：基于 Rust 核心的 Pydantic 原生异步 ORM](https://github.com/mr-fatalyst/oxyde) (Hacker News)
  Oxyde 是一款专为 FastAPI 设计的 ORM，核心由 Rust 编写，支持 Pydantic 原生模型。它解决了开发者重复定义模型的痛点，提供了高性能的异步数据库操作体验。

- [GitClassic：轻量级 GitHub 客户端](https://gitclassic.com) (Hacker News)
  GitClassic 是一个极简的 GitHub Web 客户端，页面大小仅 14KB。它支持 Issues、PR 及差异对比，为追求速度和纯净体验的开发者提供了一个替代官方臃肿界面的选择。

- [打造可靠的本地托管语音助手](https://community.home-assistant.io/t/my-journey-to-a-reliable-and-enjoyable-locally-hosted-voice-assistant/944860) (Hacker News)
